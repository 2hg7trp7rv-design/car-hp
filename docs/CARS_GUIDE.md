# CARS_GUIDE（車種詳細）

最終更新: 2025-12-25

## 役割
- HERITAGEの“神話”を、実車のディテールへ落とすページ。
- 目的は **「所有の現実」と「世代差」を具体化**すること。

## 何を書くか（好まれる要素）
- 世代差（どこで思想が変わったか）
- 壊れ方・弱点の“理由”（単なる箇条書きにしない）
- “価値が残る理由/落ちる理由”を構造で説明
- 数字は必要な分だけ（羅列で逃げない）

## 外部導線（許可。ただし文脈一致）
- 主CTA：そのページの文脈に最も合う行動（保険/買取/用品/整備）
- 副CTA：選択肢を広げる
- ルール：押し付けない／本文と矛盾しない／計測を落とさない

## 禁止パターン
- どの車にも言える一般論で埋める
- “最高/最強”の断定だけで根拠がない
