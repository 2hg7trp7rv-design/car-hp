# GUIDE_GUIDE（実用：行動）

最終更新: 2025-12-25

## 役割
- 読者の「次に何をすべきか」を具体化する **行動のページ**。
- 目的は “不安の解消” と “意思決定の前進”。

## 好まれる要素
- 失敗しやすいポイントの先回り（費用/頻度/優先順位）
- 用語の翻訳（専門語→一般語）
- 比較軸（何で迷うべきか、どこで割り切るか）

## 外部導線（許可）
- 行動と一致していること（保険/買取/用品/整備）
- 条件：押し付けない／メリデメを添える／FAQで補強

## 禁止
- 机上の理想論（現実の制約を書かない）
- “結局人による” で逃げる（判断軸を出す）
