# COLUMN_GUIDE（論点・考察）

最終更新: 2025-12-25

## 役割
- HERITAGEとCARSの間にある **“誤解・賛否・比較の落とし穴”** を解く。
- 読後に「次に読むべき具体（CARS）or 行動（GUIDE）」へ自然に送る。

## 好まれる型
- 神話の検証（なぜ広まった／どこが誤読される）
- 比較の罠（スペック比較が意味を失う条件）
- 規制・市場・メディア（評価が作られる仕組み）

## 外部導線（許可）
- 行動に直結する文脈のときだけ（維持/保険/売却/用品）
- “結論の先”として置く（本文の途中で急に売らない）

## 禁止
- 感想文（“好き/かっこいい”で終わる）
- 論点が散る（見出しごとに結論が出ない）
