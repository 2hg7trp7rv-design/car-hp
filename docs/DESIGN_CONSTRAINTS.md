# docs/DESIGN_CONSTRAINTS

「世界観を守りつつ改善する」ための制約（破る場合は理由と影響を書いてから）。

## 変更しない（原則固定）
- サイト全体の背景色（ベースカラー）は現行維持
- 文章のトーンは “CAR BOUTIQUEのHeritage世界観” を崩さない
- 既存の機能・回遊導線・CTA導線を削らない（置換はOKでも同等以上）

## 推奨する見せ方（統一）
- 背景はそのまま、コンテンツ部分は白いカード/コンテナでリッチに見せる
- 余白/角丸/カード配置はページ内で統一（統一感を優先）
- 外部アフィリエイト誘導は「カード内に自然導線」で配置（押し付けない）

## HERITAGEの制約（特に重要）
- HERITAGE本編で「買い方」「修理店への誘導」「外部アフィリエイト直誘導」はしない
- 誘導は COLUMN / GUIDE を経由して行う（内部回遊を優先）

## 変更時のチェック
- デザイン変更は “見た目だけ” に閉じる（データ/ルーティングを巻き込まない）
- 既存のカード・棚・CTAが欠けないか確認
