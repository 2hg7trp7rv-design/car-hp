# docs/GLOSSARY

- HERITAGE: 旗艦の読み物。世界観・歴史・設計思想を深く。外部誘導は直でしない（COLUMN/GUIDE経由）。
- CARS: 車種詳細。スペック/弱点/維持/魅力を整理。関連コンテンツのハブ。
- COLUMN: “読みやすい考察/体験/不安解消”。HERITAGE→実用へ橋渡し。
- GUIDE: “具体的な悩み解決”。購入・維持・比較など。外部CTAを置く主戦場。
- related*Slugs: 内部回遊の接続。存在しないslugを入れるとリンクが壊れる。
- MonetizeKey: 収益ブロックのキー（保険/買取/用品など）を束ねるための識別子。
