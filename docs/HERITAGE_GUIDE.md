# HERITAGE_GUIDE（旗艦：遺産記事）

最終更新: 2025-12-25

## 役割
- **世界観のアンカー**。検索よりも **滞在・保存・回遊**を勝ちに行く。
- “名車紹介”ではなく、**時代の合理と設計思想を分解する読み物**。

## 禁則（絶対）
- 外部アフィリエイト/買い方/おすすめ/価格誘導を入れない
- “整備工場紹介”“おすすめサービス”の類は書かない（GUIDEに渡す）

## 推奨ボリューム
- 体感 6,000〜10,000字（長くてOK）。ただし冗長は禁止。

## 推奨構成（鉄板）
1. **当時**：何が問題で、何が求められていたか（市場/文化/規制）
2. **設計思想**：メーカーの合理（なぜその解が正しかったか）
3. **評価の変遷**：当時の評価 → 失速/再評価の理由
4. **いまの目線**：価値の残り方（神話と現実の接点）※買い方は書かない

## 良いHERITAGEの要素
- “都市伝説”を **検証**する姿勢（断定は根拠がある箇所だけ）
- 具体：型式/年式/仕様/技術の固有名詞で解像度を上げる
- 結論は “すごい” ではなく、**何が残ったのか**で締める

## 内部回遊の設計（必須）
- HERITAGE内では **問いを作って終える**
- 解決先の割り当て
  - 具体像 → CARS
  - 論点補強 → COLUMN
  - 行動 → GUIDE
